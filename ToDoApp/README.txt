# 📝 ToDoApp - Aplicación de Lista de Tareas en Xamarin.Forms

Esta es una aplicación móvil desarrollada con **Xamarin.Forms** que permite agregar, editar y eliminar tareas. Utiliza **C# para la lógica de negocio** y **SQLite** como sistema de almacenamiento local.

---

## 📱 Características

- Ver una lista de tareas.
- Agregar nuevas tareas con nombre y descripción.
- Editar o eliminar tareas existentes.
- Persistencia de datos local usando SQLite.
- Manejo básico de errores y validaciones.

---

## 🛠️ Tecnologías utilizadas

- **Xamarin.Forms** (UI multiplataforma)
- **C#** (lógica de negocio)
- **SQLite-net** (almacenamiento local)
- **Visual Studio 2022 o superior**
